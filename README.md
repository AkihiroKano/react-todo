# Todo List - Веб приложение

Простое одностраничное веб-приложение для управления задачами.

## Размещен на GitHub Pages: [Открыть ToDo](https://akihirokano.github.io/react-todo/)

## Возможности

- Добавление новых задач
- Отметка выполнения
- Подсчет невыполненных задач
- Удаление одной задачи
- Очистка всех выполненных задач

## Технологии

- React
- Vite
- GitHub Pages
